# Team Task Manager 🚀

A full-stack Team Task Management web application built with React, Node.js, Express, Prisma, and MySQL. This platform helps teams create projects, assign tasks, manage workflows, and track progress efficiently with secure role-based authentication.

---

## 🌐 Live Demo

Frontend + Backend Live URL:

https://selfless-blessing-production-38d1.up.railway.app

---

## 📌 Features

- 🔐 User Authentication (Signup/Login)
- 👥 Role-Based Access Control (Admin/Member)
- 📁 Project Management
- ✅ Task Creation & Assignment
- 📊 Task Status Tracking
- 📅 Overdue Task Monitoring
- 📈 Dashboard Analytics
- 🗂 Team Workflow Management
- 🔎 Responsive UI
